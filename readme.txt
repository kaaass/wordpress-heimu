A 黑幕(black curtain) plugin for WordPress.

== Description ==
A 黑幕(black curtain) plugin for WordPress. Heimu is a mask on the text, visible only when the mouse moves over it.

== Installation ==
1. Upload the `heimu` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
